Extensiones para VSC

  1. Open in browser: usando la combinación alt B abrir la pagina en el navegador
  2. Auto rename tag: cambia automaticamente la etiqueta e cierre cuando cambia la apertura
  3. Tag inserter: permite insertar una etiqueta a un texto previamente seleccionado presionando Alt + Shift + i
  4. Alt + Shift + ↓ permite repetir una linea
